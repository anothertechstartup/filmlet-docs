# Publishing



A callsheet can either be **published** or **unpublished**. On creation callsheets are always unpublished.

<IconHeadline iconName='globe.svg' color='red'>Published</IconHeadline> The callsheet is availible for all receivers and the Sharing URL is active.

<IconHeadline iconName='globe.svg' color='red'>Unpublished</IconHeadline> The callsheet is availible for all receivers and the Sharing URL is active.
